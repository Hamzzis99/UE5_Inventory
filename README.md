졸업작품 Helluna에 적용 할 인벤토리 시스템.
- 헬루나 프로젝트 : https://github.com/Hamzzis99/Helluna
- 원본 프로젝트 링크 : https://www.udemy.com/course/unreal-engine-5-inventory-systems/

강의 내용 -
- 인벤토리 UI 형식
- 클라이언트에서만 로직 처리
- 프록시 메시



직접 만든 기능 -
재료 줍거나 사용 시 서버와 클라이언트 재료 둘 다 조건 확인
  - 빌딩 기능 (건설 기능)
  - 크래프팅 기능 (무기 제작)
  - 재료 위젯 컴포넌트 제작
  - Virtual Inventory (인벤토리 저장 대비)

- 인벤토리 세이브 기능
